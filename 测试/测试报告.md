## 七、部署说明

### 7.1 环境要求
- JDK 17+
- Node.js 18+
- MySQL 8.0+
- Redis 7.0+

### 7.2 启动步骤

#### 后端启动
```bash
cd 学习助手问答平台/后端/backend
mvn clean install -DskipTests
cd la-application
mvn spring-boot:run
```

#### 前端启动
```bash
cd 学习助手问答平台/前端
npm install
npm run dev
```

#### 数据库初始化
```bash
mysql -u root -p < 学习助手问答平台/数据库/init.sql
```

### 7.3 访问地址
- 前端：http://localhost:3000
- 后端API：http://localhost:8080
- Swagger文档：http://localhost:8080/swagger-ui.html
